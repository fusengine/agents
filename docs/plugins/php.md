# fuse-php

Expert modern, framework-agnostic PHP — libraries, standalone Symfony components, Slim / API-first apps, and CLI tools on PHP 8.5 with PER Coding Style 3.0 and PHPStan.

## Agents

| Agent | Description |
|-------|-------------|
| `php-expert` | Non-Laravel PHP development (PHP 8.5 / 8.4, PER-CS 3.0, PHPStan) |

## Skills

| Skill | Description |
|-------|-------------|
| `php-language-modern` | Property hooks, asymmetric visibility, lazy objects, pipe operator, attributes |
| `php-standards` | PER Coding Style 3.0, PSR-1/PSR-4/PSR-12, autoloading |
| `php-quality-tooling` | PHPStan levels, php-cs-fixer / PHP_CodeSniffer, Rector, CI |
| `php-testing` | PHPUnit (attributes-only) or Pest, data providers, mocking |
| `php-http-psr` | PSR-7/PSR-15/PSR-17/PSR-18, Slim, standalone Symfony HTTP |
| `php-ecosystem-reference` | Composer, common libraries, framework boundaries |

SOLID principles are provided by the shared `fuse-laravel:solid-php` skill.

## MCP Servers

- Context7 (documentation)
- Exa (web search)

## SOLID Rules

- Files < 100 lines
- Interfaces in `src/Contract/`
- PHPDoc mandatory
- Laravel projects (`composer.json` + `artisan`) use `laravel-expert` instead
