# fuse-rust

Expert safe, idiomatic Rust — libraries, CLI tools, async services, and web backends on Rust 1.96+ (2024 edition) with an ownership-first mindset and clippy-clean code.

## Agents

| Agent | Description |
|-------|-------------|
| `rust-expert` | Rust development (Rust 1.96+, 2024 edition, tokio / axum) |

## Skills

| Skill | Description |
|-------|-------------|
| `rust-core-language` | Ownership, borrowing, lifetimes, traits, generics, let chains |
| `rust-error-handling` | `Result`/`Option`, `?`, `thiserror` / `anyhow`, fallible boundaries |
| `rust-async-concurrency` | `async`/`.await`, tokio, channels, `Send`/`Sync`, cancellation |
| `rust-web-backend` | axum, tower middleware, extractors, state |
| `rust-testing-quality` | Unit/integration tests, doctests, clippy gates, benches |
| `rust-tooling-cicd` | cargo, workspaces, features, `rustfmt`, release profiles, CI |
| `rust-ecosystem-crates` | Crate selection, `serde`, dependency boundaries |

SOLID principles are provided by the shared `fuse-solid:solid-rust` skill.

## MCP Servers

- Context7 (documentation)

## SOLID Rules

- Files < 100 lines
- Traits separated per module
- Modular architecture (one contract per module)
- Never hand-edit `Cargo.lock`
