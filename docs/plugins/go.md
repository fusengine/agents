# fuse-go

Expert simple, idiomatic Go — CLI tools, libraries, concurrent systems, and backend services on Go 1.26+ with a small-interfaces mindset and golangci-lint-clean code.

## Agents

| Agent | Description |
|-------|-------------|
| `go-expert` | Go development (Go 1.26+, concurrency, golangci-lint v2 / govulncheck) |

## Skills

| Skill | Description |
|-------|-------------|
| `go-core-idioms` | Error handling (`errors.Is`/`As`), generics, slices/maps, `defer` |
| `go-concurrency` | Goroutines, channels, `select`, `context`, `errgroup` |
| `go-architecture` | Package/module architecture, `internal/`, backend services |
| `go-testing-quality` | Table-driven tests, `go test -race`, fuzzing, benchmarks |
| `go-tooling-security` | `go vet`, `golangci-lint` v2, `govulncheck`, `go mod` |

SOLID principles are provided by the shared `fuse-solid:solid-go` skill.

## SOLID Rules

- Files < 100 lines
- Small interfaces, one concern per package
- Modular architecture (`internal/` boundaries)
- Never hand-edit `go.sum`
