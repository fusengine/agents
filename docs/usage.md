# Usage Guide

## Quick Start

### Install a Plugin

```bash
claude plugins install fuse:ai-pilot
```

### Use Commands

Commands are invoked with a `/` prefix:

```
/apex          # APEX methodology workflow
/commit        # Smart conventional commit
/epct          # Explore-Plan-Code-Test workflow
```

### Use Skills

Skills are invoked with a `/` prefix:

```
/research      # Technical research methodology
/seo           # SEO optimization
/laravel-auth  # Laravel authentication docs
```

## Common Workflows

### 1. APEX Development Workflow

```
/apex
```

Systematic Analyze-Plan-Execute-eXamine approach for intelligent development.

### 2. Smart Git Commits

```
/commit
```

Analyzes changes and creates semantic commit messages.

### 3. Laravel Development

```
/laravel-architecture
/laravel-eloquent
/laravel-api
```

Access complete Laravel 12 documentation.

### 4. Next.js Development

Install the plugin:
```bash
claude plugins install fuse:nextjs
```

Use the skills:
```
/nextjs-16
/prisma-7
/better-auth
```

### 5. Swift/SwiftUI Development

```
/swiftui-components
/swift-concurrency
/apple-platforms
```

## Tips

1. **Combine skills**: Use multiple skills in one session for comprehensive context
2. **Use agents**: For complex tasks, agents provide specialized capabilities
3. **SOLID validation**: Enable `fuse:solid` for automatic code quality checks
