# NAP Citations - Consistency and Management

**NAP = Name, Address, Phone. Foundation of local SEO.**

## Why NAP Consistency is Critical

- Google uses citations to **verify legitimacy**
- Inconsistencies = mixed signals = **ranking drop**
- Direct impact on **Local Pack** and **Google Maps**
- Essential for **voice search** ("Ok Google, pizza near me")

---

## Standardized NAP Format

### Golden Rule: IDENTICAL everywhere

```text
âœ… Correct:
   Name: ABC Plumbing
   Address: 123 Main Street, New York, NY 10001
   Phone: (212) 555-1234

âŒ Incorrect (variations):
   Name: ABC Plumbing LLC / A.B.C. Plumbing / ABC-Plumbing
   Address: 123 main st / 123 Main St. / 123, Main Street
   Phone: 2125551234 / +1 212 555 1234 / 212.555.1234
```

### Standardization Checklist

| Element | Decision to Make |
| --- | --- |
| **Name** | With or without legal form (LLC, Inc) |
| **Street** | "Street" or "St." - choose ONE format |
| **State** | Full name or abbreviation |
| **Phone** | Format: (XXX) XXX-XXXX |
| **Website** | With or without www, with or without trailing / |

## Structured vs Unstructured Citations

### Structured (Directories)
- Standard NAP format
- Categories, hours, photos
- **High priority**

### Unstructured (Mentions)
- Local blogs, press, partners
- Mention of name + city
- **SEO bonus** if linked

**Unstructured mention example:**
> "ABC Plumbing, located in Manhattan's Upper East Side, won the award..."

## Special Cases

### Relocation

1. Update GBP FIRST
2. Wait for Google validation
3. Update Tier 1 (48h)
4. Update Tier 2 (1 week)
5. Scan after 30 days

### Name Change

1. Same process as relocation
2. Keep old name in description (transition)
3. Redirect old site to new
4. Monitor automatic citations

### Multi-Location

- ONE GBP profile per physical address
- Unique NAP per location
- Distinct local pages on website
