# Breadcrumb Schema

**JSON-LD for breadcrumb navigation (improves SERP display).**

```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://example.com"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Blog",
      "item": "https://example.com/blog"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "SEO Guide 2026",
      "item": "https://example.com/blog/seo-guide-2026"
    }
  ]
}
```
