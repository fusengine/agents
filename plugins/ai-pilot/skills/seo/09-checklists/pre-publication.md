# Pre-Publication SEO Checklist

**Verify before publishing content.**

## On-Page SEO

- [ ] Title tag 50-60 chars with primary keyword
- [ ] Meta description 120-155 chars with CTA
- [ ] H1 unique, contains primary keyword
- [ ] Hn hierarchy (no skipping H2â†’H4)
- [ ] URL SEO-friendly (<75 chars, lowercase, hyphens)
- [ ] Images optimized (WebP, lazy load, <200KB)
- [ ] Alt text on ALL images (descriptive, <125 chars)
- [ ] Internal links 3-5 per 1000 words
- [ ] External links 2-3 to authoritative sources
- [ ] Canonical tag present (self-referencing)

---

## Open Graph & Twitter

- [ ] og:title (60-90 chars)
- [ ] og:description (150-200 chars)
- [ ] og:image (1200x630px)
- [ ] og:url (canonical URL)
- [ ] twitter:card (summary_large_image)
- [ ] twitter:image (1200x628px)

---

## Structured Data

- [ ] Schema.org JSON-LD added
- [ ] Article schema (author, publisher, dates)
- [ ] FAQ schema (if applicable)
- [ ] Breadcrumb schema
- [ ] Validated with Rich Results Test

---

## Content Quality

- [ ] Primary keyword density 1-2%
- [ ] Secondary keywords included naturally
- [ ] First 100 words answer query (GEO)
- [ ] Statistics cited with sources
- [ ] Expert quotes included
- [ ] Published/updated date visible
- [ ] Author byline with link to bio

---

## Technical

- [ ] Page speed <3 seconds
- [ ] Mobile-responsive
- [ ] HTTPS enabled
- [ ] No broken links (404s)
- [ ] Robots meta tag correct
