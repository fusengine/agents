# Technical SEO Audit Checklist

**Comprehensive site health check.**

## Crawlability

- [ ] Robots.txt accessible (/robots.txt)
- [ ] Sitemap.xml submitted to Search Console
- [ ] No orphan pages (linked from somewhere)
- [ ] Max 3 clicks from homepage to any page
- [ ] No redirect chains (max 1 redirect)
- [ ] No 404 errors on important pages

---

## Indexation

- [ ] Site indexed (site:example.com in Google)
- [ ] No unintentional noindex tags
- [ ] Canonical tags correct (no conflicts)
- [ ] Duplicate content resolved
- [ ] Pagination handled (rel=prev/next or canonical)

---

## Performance

- [ ] LCP <2.5 seconds
- [ ] INP <200 milliseconds
- [ ] CLS <0.1
- [ ] TTFB <600 milliseconds
- [ ] Page size <3MB
- [ ] Images optimized (WebP format)

---

## Security

- [ ] HTTPS enabled (SSL certificate)
- [ ] No mixed content warnings
- [ ] Security headers (HSTS, CSP)

---

## Structured Data

- [ ] Schema.org implemented
- [ ] No errors in Rich Results Test
- [ ] Article schema on blog posts
- [ ] Organization schema on homepage
- [ ] Breadcrumb schema site-wide
