# Chore Commit

## Current State

!`git status`

## Changes

!`git diff --stat`

## Instructions

Create a maintenance/chore commit.

1. Stage all changes: `git add -A`
2. Determine scope from files:
   - `package.json` → `deps`
   - Config files → `config`
   - Linting → `lint`
3. Generate commit message:
   - If `$ARGUMENTS` provided: `chore(scope): $ARGUMENTS`
   - If empty: generate description
4. Format: `chore(scope): imperative description`
5. **NO AI SIGNATURE**: Never add Co-authored-by, Generated by, or any AI mention
6. Ask confirmation before committing
7. Execute and show result
